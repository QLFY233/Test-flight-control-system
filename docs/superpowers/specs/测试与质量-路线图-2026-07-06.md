### 1.2 验收门槛(先导)

- A 单元 + B 纯函数单元测试全绿。
- `S0`~`S7`(后端 B spec §11)逐阶段通过才进下一阶段。
- 求解器对 `field.yaml` 默认配置(box/cylinder/sphere/plane)碰撞校验:合法轨迹通过、撞墙轨迹 reject(单测覆盖)。
- msgpack 跨 venv-A(pip 最新)/venv-B(0.6.2)`use_bin_type=True` 帧互解(S0 验证项)。
- 前端不强制自动化;先导以手动验收 + 浏览器 DevTools 为主(开发计划各 P 阶段验收清单),Playwright 后补。

### 1.3 CI(可选,先导不强制定)

- 仓库可选 `.github/workflows/test.yml`:在 Ubuntu 20.04 容器内跑 A 单元(py3.10)+ B 纯函数(py3.8,不依赖 ROS),不计 ROS/gazebo。
- 先导阶段允许手动跑;阶段三联调后引入 CI。
- 测试审查 skill(原始设计稿"自动测试审查"):用 todo/test-driven-development 型流程,不自动拒绝未覆盖 PR,先导仅提示。

### 1.4 测试目录约定

```
backend-A/tests/        # pytest,纯 Python(不连 ROS/讯飞/DeepSeek,mock 外部)
backend-B/tests/        # pytest,纯函数子集(collision/trajectory/frames);rospy 相关用 rostest 在联调跑
```

- spec(`docs/superpowers/specs/`)是"设计权威",`status.md` 是"进度镜像",不重复设计内容,只记进度/阻塞/下一步。

### 3.1 开放式接口规范(独立 spec,G2)

- **来源**:研究内容 5 / 预期产出 4"开放式接口规范文档"。
- **现状**:散落在总体架构 §3.7、A spec §3.2/§3.5、B spec §2.1、A-B 接口冻结 §1-5。无独立文档供"后续新模型/程序快速集成"。
- **任务**:抽成独立 spec `docs/superpowers/specs/开放式接口规范.md`,内容:
  - 统一消息格式(`{schema_version, from, to, msg_type, call_id, tool, args, payload, ts}`)与 `msg_type`/`tool` 语义。
  - 组件接口契约(`TrajectoryTranslator`/`AnalyticsTool`/`Detector`/`SetpointAdapter`)与注册表。
  - 新组件接入步骤(实现接口 → 静态注册 → 总线路由)。
  - 动态发现/热加载演进路径(先导不做,留接口)。
- **前置**:A/B/前端先导打通(S3+),组件契约经实战验证后再固化规范文档。
- **启动时机**:阶段三联调稳定后。

### 3.2 PX4 阶段 2 详细 spec(G4)

- **来源**:后端 B spec §11 S8 + §12 仅核实表,无详细 spec。
- **任务**:`docs/superpowers/specs/PX4-阶段2-design.md`,内容:
  - PX4 SITL 跑法(headless):`make px4_sitl jmavsim` / `HEADLESS=1 make px4_sitl gz_x500` / `make px4_sitl none_iris`(已核实,B spec §12)。
  - mavros 1.20.1 配置 + offboard 模式切换时序(setpoint stream ≥2Hz 数秒再切 offboard)。
  - `rosbridge/topics.py` phase=2 切换:`/mavros/...`,setpoint 用 `mavros/setpoint_raw/local`(`mavros_msgs/PositionTarget` + `type_mask` 自由度位掩码,已核实 B spec §5.1/§5.2)。
  - `type_mask` 取值表(位置/速度/加速度/yaw/yaw-rate 位掩码对照)。
  - `SetpointAdapter.Phase2` 实现细节、首帧前 stream、offboard 退出兜底。
  - `pymavlink`(pip,apt 不提供)装法 + mavros apt 包名(`ros-noetic-mavros*` 1.20.1)。
- **前置**:阶段1 假无人机链路全通(S0~S7)+ PX4 SITL 本机环境就绪。
- **启动时机**:阶段三联调尾声 / 阶段四外场前。
